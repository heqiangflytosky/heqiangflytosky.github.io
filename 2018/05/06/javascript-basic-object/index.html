介绍 JavaScritp 的类和对象

概述

Javascript 是一种基于对象的语言，你遇到的所有东西几乎都是对象。但是，它又不是一种真正的面向对象编程（OOP）语言，因为它的语法中没有 class（类），没有 extend 关键字，这也就导致传统的面向对象编程方法无法直接使用。
